# Hero Images Setup Instructions

## Required Hero Images

Please save the three provided hero images in the `public` folder with the following names:

1. **hero-1.jpg** - Modern lakeside villa (floating house with warm lighting at night)
2. **hero-2.jpg** - Mountain snow resort (wooden chalets with snow-covered mountains)
3. **hero-3.jpg** - Urban rooftop lounge (city skyline view with modern terrace)

## Steps:

1. Save the three images from the attachments/context
2. Rename them to:
   - `hero-1.jpg`
   - `hero-2.jpg`
   - `hero-3.jpg`
3. Place them in: `f:\Hotello\frontend\public\`

## File Structure:
```
frontend/
  public/
    hero-1.jpg  ← Modern lakeside villa
    hero-2.jpg  ← Mountain snow resort
    hero-3.jpg  ← Urban rooftop lounge
    bg-hero.jpg (existing)
    ...
```

## Image Specifications:
- Format: JPG (or PNG/WebP)
- Recommended size: 1920x1080px (Full HD)
- Aspect ratio: 16:9
- File size: Optimized for web (under 500KB per image recommended)

## Features:
- Automatic carousel rotation (every 5 seconds)
- Smooth transitions with fade effects
- Navigation arrows for manual control
- Indicator dots showing current slide
- Auto-play pauses on hover
- Optimized loading (eager loading for hero images)
